Glossário de Tecnologia da Informação

Conheça conceitos essenciais utilizados no desenvolvimento e gerenciamento de tecnologias.

Sobre o Glossário

A Tecnologia da Informação (TI) é uma área responsável pelo uso de computadores, sistemas, redes e softwares para criar, armazenar, processar e transmitir informações. Este glossário apresenta alguns dos principais termos utilizados por estudantes e profissionais da área de tecnologia.

Termos de Tecnologia da Informação

HTML

HyperText Markup Language

HTML é uma linguagem de marcação utilizada para criar a estrutura de páginas da internet. Ela define elementos como títulos, textos, imagens, links, formulários e tabelas. O HTML funciona como a base de um site, organizando todo o conteúdo que será exibido pelo navegador.

Exemplo de uso: Criação de páginas web, blogs, sistemas online e sites institucionais.

CSS

Cascading Style Sheets

CSS é uma linguagem utilizada para estilizar páginas desenvolvidas com HTML. Com CSS é possível alterar cores, fontes, tamanhos, espaçamentos, animações e criar layouts responsivos.

Exemplo de uso: Design de sites, aplicativos web e interfaces modernas.

JavaScript

Linguagem de Programação

JavaScript é uma linguagem responsável por adicionar interatividade aos sites. Com ela é possível criar menus dinâmicos, validações de formulários, animações e sistemas completos.

Exemplo de uso: Aplicações web, jogos, sistemas online e aplicativos.

Bootstrap

Framework Front-end

Bootstrap é um framework utilizado para facilitar o desenvolvimento de interfaces. Ele possui componentes prontos como botões, menus, cards e sistemas de organização de páginas.

Importância: Ajuda desenvolvedores a criarem sites mais rápidos e responsivos.

API

Application Programming Interface

API é uma tecnologia que permite a comunicação entre diferentes sistemas. Ela funciona como uma ponte permitindo que aplicações troquem informações de forma organizada.

Exemplo: Aplicativos bancários consultando dados de servidores.

Banco de Dados

Armazenamento de informações

Banco de dados é um sistema utilizado para armazenar, organizar e consultar informações. Ele é essencial para sistemas que precisam guardar dados de usuários, produtos, pagamentos e registros.

Exemplo: MySQL, PostgreSQL e Oracle Database.

Servidor

Infraestrutura de Tecnologia

Servidor é um computador ou sistema responsável por fornecer serviços para outros dispositivos conectados. Ele pode armazenar arquivos, hospedar sites ou executar aplicações.

Exemplo: Servidores de empresas, bancos e plataformas online.

Hospedagem

Serviço de publicação online

Hospedagem é o serviço responsável por disponibilizar um site ou sistema na internet. Ela utiliza servidores para guardar arquivos e permitir acesso através dos navegadores.

Exemplo: Hospedagem de sites profissionais e lojas virtuais.

Git

Controle de versão

Git é uma ferramenta utilizada para controlar alterações em códigos. Ele permite que desenvolvedores acompanhem versões de projetos, voltem alterações e trabalhem em equipe.

Uso: Muito utilizado no desenvolvimento de softwares.

GitHub

Plataforma de projetos

GitHub é uma plataforma online baseada no Git. Ela permite armazenar códigos, compartilhar projetos e colaborar com outros desenvolvedores.

Importância: Muito utilizado por empresas e programadores no mundo inteiro.

Importância da Tecnologia da Informação

A TI é fundamental para empresas, governos e pessoas. Ela possibilita comunicação rápida, automação de processos, segurança de dados e criação de novas soluções digitais. Profissionais de TI trabalham com desenvolvimento de sistemas, redes, bancos de dados, segurança e inovação tecnológica.
