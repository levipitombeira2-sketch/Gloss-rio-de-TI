Glossário de Tecnologia da Informação

Conheça alguns dos principais conceitos utilizados na área de TI.

Introdução

A Tecnologia da Informação está presente em praticamente todos os setores da sociedade. Este glossário apresenta alguns dos termos mais importantes da área, ajudando estudantes e iniciantes a compreender conceitos fundamentais do desenvolvimento web e da informática.

HTML

Linguagem de marcação utilizada para estruturar páginas web.

CSS

Linguagem responsável pela aparência e estilização das páginas.

JavaScript

Linguagem de programação que adiciona interatividade aos sites.

Bootstrap

Framework CSS que facilita a criação de interfaces responsivas.

API

Conjunto de regras que permite a comunicação entre sistemas.

Banco de Dados

Local onde informações são armazenadas e organizadas.

Servidor

Computador responsável por fornecer recursos e serviços na rede.

Hospedagem

Serviço que mantém um site disponível para acesso na internet.

Git

Sistema de controle de versão utilizado por desenvolvedores.

GitHub

Plataforma que hospeda projetos e permite colaboração utilizando Git.

Resumo das Tecnologias

Tecnologia Categoria Função
HTML Estrutura Organização do conteúdo
CSS Estilo Visual da página
JavaScript Programação Interatividade
Git Versionamento Controle de alterações
