Glossário de Tecnologia da Informação

Conhecendo os principais conceitos do mundo da tecnologia.

Sobre o Glossário

A Tecnologia da Informação está presente no desenvolvimento de sistemas, aplicativos, sites e gerenciamento de dados. Este glossário apresenta alguns termos importantes utilizados por profissionais da área de TI.

Principais Termos de TI

HTML

Linguagem usada para criar a estrutura das páginas web.

CSS

Responsável pelo design, cores e organização visual dos sites.

JavaScript

Linguagem utilizada para criar interações e funcionalidades.

Bootstrap

Framework que facilita a criação de páginas responsivas.

API

Permite comunicação entre diferentes sistemas.

Banco de Dados

Organiza e armazena informações importantes.

Servidor

Computador responsável por fornecer serviços na rede.

Hospedagem

Serviço que mantém sites disponíveis na internet.

Git

Ferramenta para controle de versões de projetos.

GitHub

Plataforma usada para armazenar e compartilhar códigos.

Áreas da Tecnologia

Front-end

Parte visual dos sistemas, responsável pela interface.

Back-end

Responsável pela lógica e funcionamento interno.

Cloud Computing

Uso de servidores e serviços pela internet.
